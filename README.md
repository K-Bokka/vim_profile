# vim_profile

```shell
# clone this repo to ~/.vim
$ cd
$ git clone git@github.com:K-Bokka/vim_profile.git .vim

# Install dein vim. see: https://github.com/Shougo/dein.vim
$ curl https://raw.githubusercontent.com/Shougo/dein.vim/master/bin/installer.sh > installer.sh
$ sh ./installer.sh ~/.vim/bundles
$ rm ./installer.sh

# Open vim
$ vi
```
